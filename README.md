# Superstore Sales Analysis using SQL

## 📌 Project Overview
This project analyzes the Sample Superstore dataset using SQL to understand sales performance, profitability, and loss-making areas.

---

## 📊 Dataset
- Dataset: Sample Superstore
- Rows: 9994
- Tool Used: SQLiteStudio

---

## 🎯 Business Questions Solved
1. What is the total sales and total profit?
2. Which category has the highest sales?
3. Which category has the highest profit?
4. Which region has the highest profit?
5. Which sub-category has the highest profit?
6. Which sub-categories are loss-making?

---

## 📈 Key Insights
- Technology generated the highest sales and profit  
- West region was the most profitable  
- Copiers had the highest profit among sub-categories  
- Tables, Bookcases, and Supplies were loss-making  

---

## 🛠️ SQL Skills Used
- SUM()
- GROUP BY
- ORDER BY
- HAVING
- Data Aggregation
